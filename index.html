Yelp-Camp

A E-Commerce Website

A decently big project, it allows users to authenticate and login, while also acting as e-commerce website for Campgrounds. It uses Express, NodeJS and MongoDB(Mongo Atlas) to implement the backend part of the app. Cloudinary is used for image storage, helmet, sanitize-html and express-mongo-sanitize have been used to implement the basic security. Passport is used to implement authentication. Mapbox is used for showing maps and displaying campgrounds on the map along with geocoding the locations. HTML, CSS, Bootstrap and JavaScript have been used to implement the frontend part of the application, along with the use of various features like DOM. And Heroku has been used for deployment of the app.
